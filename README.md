# Projet Charte Template Deploiement Service & Donnée Windows Server
- (En cours de développement)

### Info :
- Une charte de Template IaC en Powershell pour deployer un/des service/s dans un/des serveur/s Windows
- Dans la même philosophie que Helm Chart et Ansible mais avec les service Windows Server en version simplifié


## Remarque :
- Les seule donnée à modifier selon les besoins seront juste les élément du dossier "data"
- Si besoin, s'aider du dossier "data_template"
